# Портфолио лабораторных работ

Добро пожаловать на сайт-портфолио лабораторных работ по курсу **Python**.

## О сайте

Здесь собраны отчёты по лабораторным работам семестрового курса. Каждая работа содержит:

- цель работы;
- описание задания;
- код с пояснениями;
- выводы.
